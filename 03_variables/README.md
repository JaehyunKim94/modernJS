# 03 Variables & Value

숫자, 문자열, 객체 등 자바스크립트가 다루는 값(데이터 타입)과 값을 저장하기 위한 변수를 학습

## 3.1 변수

변수는 값을 담기위해 이름을 붙인 상자입니다.

```javascript
var x;			// 변수 선언
console.log(x);	// undefined
x = 2;			// 변수에 값을 대입
console.log(x);	// 2
var y = 6;		// 변수 선언 & 초기값 설정
var a = 1, b = 2, c = 3;	// 여러 개의 변수에 대해 초기값 설정을 쉼표로 구분
```
